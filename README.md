# TG-BOT-Artem-2 — клубная воронка (club3)

Telegram-бот на **aiogram 3**. Логика портирована из `tertiary_bot_standalone`
(club3-сценарий) и привязана к каналу этого проекта. Квиз, автопостинг,
юзерботы, AI-генерация и дожимная воронка удалены.

## Режимы работы

При первом запуске мастер настройки предлагает выбрать режим (`BOT_MODE`):

- **Вариант 1** (`BOT_MODE=1`) — воронка стартует сразу по `/start`, подписка
  на канал предлагается в конце (кнопкой на шаге 3).
- **Вариант 2** (`BOT_MODE=2`) — бот ждёт вступления в канал и пишет первым
  сразу после заявки на вступление (как в `tertiary_bot_standalone`).

Режим можно изменить в любой момент в `.env`.

## Сценарий воронки

1. **`/start`** — бот просит подписаться на канал и дождаться подтверждения
   заявки (кнопка-ссылка на канал). *(если `CLUB_FLOW_REQUIRES_JOIN=true`)*
2. **Заявка на вступление** в канал (`chat_join_request`) — бот её одобряет
   *(если `AUTO_APPROVE_JOIN=true`)* и запускает воронку в личке.
3. **Стартовое фото + кнопка «Я не робот»**.
4. **Шаг 2** — «Как заработать в 2026».
5. **Шаг 3** — кнопки «ГОЛОС КЛУБА» (ссылка на менеджера) и «СМОТРЕТЬ ОТЗЫВЫ».
6. **«СМОТРЕТЬ ОТЗЫВЫ»** — альбом с отзывами.

Есть спец-диплинк `/start club3_not_robot` — сразу к шагу 2.

## Требования к каналу

- Бот — **администратор** канала.
- Инвайт-ссылка — типа **«заявка на вступление»** (request-to-join), иначе
  событие `chat_join_request` не приходит.
- В `.env` указан корректный `CHANNEL_ID` (с `-100...`).

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

При **первом запуске** (нет `.env` или не заполнен `BOT_TOKEN`) в терминале
автоматически откроется **мастер настройки** — он спросит токен, ID админа,
данные канала, менеджера, флаги поведения и Keitaro, и сам создаст `.env`.

Можно настроить и вручную: `cp .env.example .env` и заполнить значения.

Или через Docker (мастер пропускается — заполните `.env` заранее):

```bash
cp .env.example .env
docker compose up -d --build
```

## Сборка в .exe (Windows)

```bash
python build_bot.py        # или двойной клик по build_bot.bat
```

Готовый `club_bot.exe` появится в папке `dist/` (медиа упакованы внутрь).
При первом запуске `.exe` сам откроет мастер настройки и создаст `.env` рядом.

## Настройка

Тексты, подписи кнопок и медиа настраиваются через `.env` (см. `.env.example`)
или прямо в `config.py`. Стартовое фото и отзывы лежат в `media/`.

## Команды

- `/admin` — статистика воронки (только для `ADMIN_ID`).

## Структура

| Файл | Назначение |
|------|------------|
| `main.py` | хендлеры и логика воронки |
| `config.py` | конфигурация (канал, тексты, медиа, флаги) |
| `keyboards.py` | клавиатуры |
| `database.py` | SQLite-хранилище состояния |
| `postback.py` | Keitaro PostBack |
| `build_bot.py` / `build_bot.bat` | сборка в `.exe` |
| `media/` | стартовое фото и отзывы |
